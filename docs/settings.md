# Settings Panel Guide

The Settings panel has been redesigned with a category-based menu system for easier navigation.

## Opening Settings
Click the gear icon in the top-left corner of the application.

## Menu Structure

### AI & Language Category
- **Language Settings**: UI translation and response language
- **LLM / OpenRouter**: Configure custom LLM models
- **Persona System Prompt**: Define character personality
- **Memory Settings**: Conversation memory and summarization
- **Context Settings**: Time, battery, and context awareness
- **Automation Settings**: Auto-response and triggers

### Models & Visuals Category
- **Model Settings**: Select and manage Live2D models
- **Display Settings**: Panel positions and visibility
- **Background Image**: Set and manage backgrounds

### Audio Category
- **Voice Settings**: TTS configuration
- **Audio Settings**: Volume and sound effects

### System Category
- **Preferences**: General application preferences
- **Debug Settings**: Logging and troubleshooting options

### Additional Sections
- **Links**: External links and resources
- **Help & Tutorial**: Interactive tutorial and documentation

## Search Functionality

Use the search bar at the top of the Settings panel to quickly find specific settings by name.

## Submenus

Each menu item opens a submenu with detailed options. Use the back button (‹) to return to the main menu.
